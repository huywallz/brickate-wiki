Returns the current transparency state of a brick.

```
bk.brick.getAlpha( id )
```

## Parameters
**<code>[bk.object](?p=bk.object) id</code>**
> The unique identifier of the brick object.

## Returns
**<code>number alpha</code>**
> The current alpha value (typically `0.0` for fully transparent to `1.0` for fully opaque).
