Checks if two colors are identical.

```
bk.color.equals( color1, color2 )
```

## Parameters
**<code>[bk.color](?p=bk.color) color1</code>**
> The first color table to compare.

**<code>[bk.color](?p=bk.color) color2</code>**
> The second color table to compare.

## Returns
**<code>boolean matches</code>**
> `true` if all components are exactly the same, `false` otherwise.
