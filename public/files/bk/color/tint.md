Multiplies one color by another to create a tinted effect.

```
bk.color.tint( color, tint )
```

## Parameters
**<code>[bk.color](?p=bk.color) color</code>**
> The base color table to be tinted.

**<code>[bk.color](?p=bk.color) tint</code>**
> The color table used as the tinting layer.

## Returns
**<code>[bk.color](?p=bk.color) color</code>**
> A new color table representing the multiplicative blend of both colors.
