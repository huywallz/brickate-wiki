Adjusts the brightness of a color.

```
bk.color.brightness( color, amount )
```

## Parameters
**<code>[bk.color](?p=bk.color) color</code>**
> The color table to modify.

**<code>number amount</code>**
> The target brightness multiplier (typically `0.0` to `1.0`).

## Returns
**<code>[bk.color](?p=bk.color) color</code>**
> A new color table with the adjusted brightness value.
