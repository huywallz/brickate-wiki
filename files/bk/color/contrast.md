Adjusts the contrast level of a color.

```
bk.color.contrast( color, amount )
```

## Parameters
**<code>[bk.color](?p=bk.color) color</code>**
> The color table to modify.

**<code>number amount</code>**
> The target contrast value (values above `0` increase contrast, while negative values decrease it).

## Returns
**<code>[bk.color](?p=bk.color) color</code>**
> A new color table with the adjusted contrast applied.
