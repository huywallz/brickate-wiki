Creates a color from a single integer value.

```
bk.color.fromInt( hex )
```

## Parameters
**<code>number hex</code>**
> The integer or hexadecimal value representing the color.

> Range: `0` (Black) to `16777215` (White).

## Returns
**<code>[bk.color](?p=bk.color) color</code>**
> A new color table derived from the integer.
